# backend

## API Endpoints
| Endpoint     | Params                           | Returns   | Description                        |
|:-------------|:---------------------------------|:----------|:-----------------------------------|
|POST /get_text|String image: Base64 encoded image|String text|Text that is recognized in the image|

## Usage
`pip install -r requirements.txt`  
`export FLASK_APP=app.py`  
`flask run`
